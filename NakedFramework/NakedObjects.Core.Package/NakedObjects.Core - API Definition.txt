This document defines the API for the NakedObjects.Framework package (as used for Semantic Versioning).

The API for the framework consists of:

- The set of domain model programming conventions (recognised methods, attributes, and types) recognised by the framework, 
including but not limited to those defined in the NakedObjects.ProgrammingModel package on which this depends.
